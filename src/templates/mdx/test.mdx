# Example Markdown Table

Below is an example of a Markdown table with different alignments and other Markdown elements inside:

| **Feature**         | **Description**                          | **Syntax**           |
|:--------------------|:----------------------------------------:|----------------------:|
| **Bold**            | _Emphasize important text_              | `**text**`           |
| _Italic_            | Makes text _slanted_                    | `_text_`             |
| [Link](https://www.example.com) | Link to an external page             | `[text](URL)`        |
| Inline `code`       | Show inline code snippets               | `` `code` ``         |
| List                | - Item 1<br />- Item 2                    | `- Item 1`<br />`- Item 2` |
| Centered Text       | Centered content                        | `:-:`                |
| Right-aligned       | Aligns text to the right                | `--:`                |
